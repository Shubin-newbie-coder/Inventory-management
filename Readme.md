# ລະບົບຈັດການຄັງສິນຄ້າ / Inventory Management System

## Tech Stack
- Python
- PostgreSQL
- SQL
- psycopg2

## Features
- ເພິ່ມສິນຄ້າ
- ສະແດງສິນຄ້າ
- ລຶບສິນຄ້າ

## Setup
1. Create database `inventory_db`
2. Run `schema.sql`
3. Install dependencies
4. Run `python main.py`
